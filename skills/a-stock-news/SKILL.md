# A 股舆情监控工具包

## 核心功能
通过抓取财经新闻、社区讨论热度及情绪倾向，辅助判断个股/板块的**市场关注度**与**情绪拐点**。

## 数据获取与 API

### 1. 7x24 小时快讯 (Telegraph)
实时获取市场重大消息。
```python
import akshare as ak

# 财联社/东方财富 7x24 快讯
df_news = ak.stock_info_global_em() 
# 或
df_cls = ak.stock_zh_a_alerts_cls() 
# 字段: 标题, 内容, 时间, 链接
```
**过滤逻辑**: 提取关键词如 "利好", "减持", "业绩", "中标", "立案调查" 等进行分类。

### 2. 股吧/社区热度 (East Money Guba / Xueqiu)
判断散户情绪 (通常是反向指标) 或大 V 观点 (正向指标)。
- **东方财富股吧**: `ak.stock_comment_em(symbol="601872")` (获取最新帖子/评论)
- **雪球**: 网页版 `https://xueqiu.com/S/SH601872` 
  - 抓取方法: `browser_navigate` + `browser_console` (提取讨论区内容)

### 3. 热度分析 (Sentiment Analysis)
- **正面信号**: 机构研报密集发布、知名大 V 看多且逻辑清晰、重大利好政策出台。
- **负面信号**: 股吧情绪极度亢奋 (全部喊涨停)、负面澄清公告、大股东减持预告。

## 舆情实战逻辑

### 消息面 + 资金面共振
1. **突发利好**: 舆情出现重大利好 + 资金大幅净流入 -> **跟随做多**。
2. **利好兑现**: 舆情发酵很久 + 股价高位 + 资金流出 -> **利好出尽是利空**。
3. **利空出尽**: 坏消息满天飞 + 股价跌不动 + 资金悄悄流入 -> **左侧布局**。

### 热度背离 (Divergence)
- **量价配合**: 股价上涨 + 舆情热度上升 = 健康趋势。
- **量价背离**: 股价创新高 + 舆情无人问津 = 警惕诱多 (或为强庄控盘)。
- **情绪极值**: 当股吧情绪达到 100% 看多时，往往是短线顶部。

## 脚本示例 (`scripts/news_sentiment.py`)
- **功能**: 
  1. 抓取财联社 7x24 快讯。
  2. 利用正则/关键词过滤出与持仓股票相关的内容。
  3. 生成 `~/a-stock/data/news_feed.json`。
- **依赖**: `akshare`, `re`, `pandas`.

## 注意事项
1. **噪音过滤**: 社区帖子 (尤其是股吧) 充满情绪化宣泄和庄家水军，**务必作为反向参考或情绪极值判断**，不要盲目信从。
2. **信息时效**: 免费 API 抓取通常有 5-15 分钟延迟，高频交易者需使用付费 Level-2 资讯。
3. **网络限制**: 东方财富/雪球对爬虫有较严格的反爬策略 (封 IP/验证码)，建议控制请求频率或使用浏览器方式。
